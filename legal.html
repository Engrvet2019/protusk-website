The Software does not collect, transmit, or share any of the following:

The Software stores your encrypted vault file locally on your device using your operating system's standard application data directory. The AI editions include a localhost-only REST API on port 7777. This API operates exclusively on your local machine (127.0.0.1) and does not transmit data over any network.
